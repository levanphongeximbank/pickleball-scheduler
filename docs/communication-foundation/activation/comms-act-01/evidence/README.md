# Evidence directory (COMMS-ACT-01)

Place filled Staging activation evidence here during **COMMS-ACT-02**.

Rules:

- No secrets (tokens, passwords, connection strings, service-role keys)
- Filenames: `STAGING_ACTIVATION_EVIDENCE_<ISO_DATE>.md` or `.json`
- Use templates from parent folder

COMMS-ACT-01 ships this directory empty of live evidence on purpose.
