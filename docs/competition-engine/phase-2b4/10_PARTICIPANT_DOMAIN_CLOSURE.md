Participant Domain:

## Meaning

Phase 2B (participant track) has a closed **foundation**: contracts, validators, DTOs, ports, format adapters, shadow mapping, and integration evidence.

This does **not** mean:

- Competition Core is Production SSOT
- Participant runtime is active
- Persistence or cutover is done
- Legacy models are retired

Owner must approve any later phase that activates runtime, persistence, or cutover.
